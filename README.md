# Приложение для обувного магазина

## Приложение для:
- Учета товаров (обуви), добавления товаров, редактирование карточек товаров (изменение описания, поставщика и т.д).
- Работы с заказами (просмотр, добавление и редактирование.
- Поиска, сортировка и фильтрации.

Есть форма для регистрация (с капчей) и авторизации (3 роли: авторизированный пользователь, менеджер и администратор).
