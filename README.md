# Приложение для обувного магазина

Приложение для учета товаров (обуви), добавления товаров, редактирование карточек товаров (изменение описания, поставщика и т.д). Работа с заказами (просмотр, добавление и редактирование. Поиск, сортировка и фильтрация. Есть регистрация (с капчей) и авторизация (3 роли: авторизированный пользователь, менеджер и администратор).
